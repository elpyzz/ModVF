# ModVF

Monorepo SaaS pour la traduction automatique de modpacks Minecraft.

## Stack
- Frontend: React 18 + TypeScript + Vite
- Styling: Tailwind CSS v4
- Routing: React Router
- State: Zustand
- Shared: Types front/back dans `packages/shared`

## Démarrage
```bash
npm install
npm run dev
```

## Structure
- `apps/web`: application frontend
- `packages/shared`: types partagés
